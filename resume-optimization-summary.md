## 📝 Resume Usage Tips

### For Job Applications:
- Customize summary for each role
- Highlight relevant projects first
- Match keywords from job description
- Keep to one page for early career

### For ATS Systems:
- Use standard section headers
- Include relevant keywords
- Avoid graphics and tables
- Save as PDF with text formatting

### File Naming:
- Use: `FirstName_LastName_Resume.pdf`
- Example: `Gowri_Shankar_Resume.pdf`
